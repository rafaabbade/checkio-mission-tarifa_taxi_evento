Como posso verificar se há um evento e ajustar os valores conforme necessário?

Você pode usar uma estrutura if para verificar se há um evento (evento == True). Caso haja um evento, multiplique a tarifa_base por 2 e a tarifa_por_km por 1,5. Caso contrário, mantenha os valores originais.

Como usar um booleano diretamente em uma condição?

Você pode usar o booleano diretamente na condição do if. Por exemplo, if evento: verifica se evento é True.
